# 고려대학교 COSE457 실전 SW 프로젝트

### 디자인 패턴을 적용한 그림판 프로그램

사용 언어 : JAVA  
라이브러리 : awt, Swing

### 주요 기능

도형 (사각형, 타원형, 선) 그리기  
도형 선택 (단일, 다중)  
도형 색 변경  
도형 크기 변경  
도형 배치 변경 (맨 앞으로 가져오기, 맨 뒤로 보내기)  
실행 취소 / 다시 실행

### 디자인 패턴

커맨드 패턴 (Command Pattern)  
팩토리 패턴 (Factory Pattern)  
싱글톤 패턴 (Singleton Pattern)  
상태 패턴 (State Pattern)  
옵저버 패턴 (Observer Pattern)  
컴포짓 패턴 (Composite Pattern)
